# CXXMessageQueue

A lock-free MPSC message queue.

This class is thread safe when used with multiple producers and a single consumer.

> [!NOTE]
> C++20 is required.

## Installation

### Swift Package Manager

Add a package dependency to https://github.com/sbooth/CXXMessageQueue in Xcode.

### Manual or Custom Build

1. Clone the [CXXMessageQueue](https://github.com/sbooth/CXXMessageQueue) repository.
2. `swift build`.
